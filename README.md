
Memento: Lost Memories is a text-based adventure game inspired by the plot of Memento. 
Players wake up with no memory and must explore a mysterious mansion, interact with NPCs, solve puzzles, and uncover memory fragments to reveal the truth about their identity.

Gameplay

Use simple text commands to navigate and interact with the environment.
Common commands include:
go [direction]      - Move to another room
take [item]         - Pick up an item
drop [item]          - Use an item
interact            - Interact with an NPC
solve               - Attempt to solve a puzzle in the current room
inventory           - View collected items
journal           - View recovered memory fragments
help                - Show available commands
quit                - Exit the game
save                - Save current progress
load                - Load saved progress


Installation

Requirements:

Java 17 or higher
A terminal or command line environment

Clone or Download the Repository:
git clone https://github.com/yourusername/memento-lost-memories.git

cd memento-lost-memories

Compile the Java Files:

javac Main/*.java Main/Commands/*.java

Run the Game:
java Main.MementoEpic
